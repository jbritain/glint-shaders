Glint

by jbritain

Glint is my shaderpack for Minecraft. It's still in development, but available to download in alpha!


click the dropdown and then 'Source code (zip)'

Bear in mind that this is an alpha, so things may be missing or broken. If you encounter issues, please let me know by submitting a Github Issue!



Want to see more? Check out #jbritains-shaderpacks in the shaderLABS Discord Server.

Features

  • LabPBR support
  • Fully forward rendered translucents even with SSR (everything has reflections)
  • PCSS 'Variable Penumbra' shadows
  • Volumetric clouds, fog, and water
  • Cloud shadows
  • Global illumination
  • Auto exposure
  • Nice bloom
More stuff is planned, such as...
  • Distant Horizons support
  • Coloured blocklight
  • Ambient occlusion
  • Depth-of-field
  • Motion blur
  • Fancy volumetric effects in the Nether and End dimensions

Compatibility

Glint requires
  • An AMD or NVIDIA GPU (Intel is untested but I do want to support it so if there are issues lmk)
  • Iris (Optifine is not supported)
  • A relatively recent version of Minecraft
  • OpenGL 4.3 support (no Apple Silicon or Raspbbery Pi)
